# Git Setup

## Adding changes to `main`

It requires a **PR** with:

1. One approval.
2. Conversation resolved.

## Auto-Merge

There's a workflow that automatically merges PRs when checks pass, except for those with the `do-not-merge` label or marked as **draft**. See [AUTO-MERGE-GUIDE.md](./AUTO-MERGE-GUIDE.md) for more details.
